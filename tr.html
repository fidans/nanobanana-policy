Uygulama: NanoBanana  •  Son güncelleme: 02.09.2025

Genel Bakış

NanoBanana, metinden görsel üretmenizi sağlar. Giriş için Firebase Authentication (Apple/Google) kullanırız. Kişisel verileri satmayız.

İşlediğimiz Veriler

  • Hesap verileri: Firebase Auth üzerinden e-posta veya Apple/Google hesap tanımlayıcıları.
  • Prompt’lar: Gönderdiğiniz metinler Firebase Cloud Function proxy üzerinden Fal’a iletilir. Sunucu tarafında prompt veya üretilen görselleri saklamayız.
  • Cihaz depolaması: Kaydetmeyi seçtiğiniz görseller, izin vermeniz halinde yerel Fotoğraflar’da tutulur.

Veri Kullanımı

  • Kullanıcı kimlik doğrulaması ve oturum sürdürülebilirliği.
  • Fal ile proxy üzerinden talep anında görsel üretimi.
  • Günlük kota bilgisinin gösterimi.

Paylaşım

Verileri yalnızca uygulamanın çalışması için gerekli hizmet sağlayıcılarla paylaşırız: Kimlik doğrulama için Firebase (Google) ve görsel üretimi için Fal. Kişisel verileri satmayız; üçüncü taraf reklam SDK’ları kullanmayız.

Saklama

Kimlik doğrulama kayıtları hesabınız var olduğu sürece Firebase’de tutulur. Sunucu tarafında üretilen görselleri saklamayız.

Silme

Hesabınızı uygulama içinden silebilirsiniz: Ayarlar → Hesabımı Sil. Bu işlem Firebase kullanıcı kaydınızı kaldırır. Fotoğraflar’a kaydettiğiniz görseller cihazınızda kalır.

Güvenlik

API anahtarları sunucu tarafında (Firebase Cloud Function proxy) tutulur. Tüm ağ trafiği HTTPS/TLS ile korunur.
